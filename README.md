Gorc
====

Description
-----------

Collected C++11 code for handling Dark Forces II assets.

Requirements
------------

* Standard C++11 compiler (Clang 3.3, GCC 4.8)
* GNU Make
* Flex 2.5
* Bison 2.5
* [Boost](http://boost.org)
* [NullUnit](http://github.com/jdmclark/nullunit)
* Dark Forces II (for running test cases)
